# Meta-S.r.l
Portale presentazioni servizi Meta Advisory

Questa cartella contiene un sito statico professionale per presentare un servizio di automazioni PowerPoint, pensato anche per clienti istituzionali come banche.

## Funzionalità incluse

- Area pubblica per descrivere chi siete e cosa offrite.
- Catalogo delle presentazioni diviso per tema.
- Sezione per visualizzazione protetta in anteprima.
- Call to action commerciale per demo e contatti.

## Come usare

1. Apri `index.html` in un browser.
2. Controlla i contenuti e sostituisci i testi con le informazioni della tua offerta.
3. Aggiorna `catalog.html` con i nomi e le descrizioni delle presentazioni.
4. Se vuoi aggiungere vere anteprime protette, sostituisci i placeholder con PDF o servizi di visualizzazione online.
